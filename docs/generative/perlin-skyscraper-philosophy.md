# 运动：雾中脊线

这场算法美学把城市当作一条会呼吸的山脊来雕刻。它不描摹建筑本体，而是让轮廓在噪声场中自发成形，仿佛风压、潮汐与时间共同塑造的地平线。真正的美来自过程本身：由精密的噪声层叠、尺度控制和节律分配构成的系统，在每一次运行中给出不同但可信的天际线。

算法的核心是层级噪声与节律阈值的协作：低频噪声塑造整体起伏，高频噪声注入细小的、像风蚀般的纹理；而稀疏的尖峰是通过精心设定的幂次与阈值生成，像在光滑轮廓中突然拔起的塔冠。这里的每一条曲线都被反复校准，仿佛由一位顶级计算美学匠人反复打磨的、经过无数轮参数试验的成果。

这不是随机扰动，而是控制中的自由。噪声强度与频率必须被精确调谐，使轮廓既有连续流动的美感，又保留细微的非均质性；每个参数都被设计成可解读的音乐旋钮，轻微改变就会带来整体气质的显著转折。如此严谨的尺度管理，体现了深厚的计算艺术功底与对算法稳定性的苛刻追求。

时间在此不是动画帧，而是隐含的结构秩序：同一种算法在不同种子下显露不同的城市气质，像同一地带在不同季风季节呈现的天际变化。这个系统像一首经过大师级优化的曲子，演奏时充满即兴，却从未失控；其细节密度与宏观平衡感来自长期的调参与严密的优化。

概念的暗线隐藏在“风廊”之中：那些不直接可见的气流与剪切层，决定了天际线的弧度与尖峰的稀疏分布。它们被嵌入噪声的层次结构里，作为微妙的方向性倾向而存在——不被直接指认，却能被熟悉城市气候的人直觉感知。这份隐秘的引用，使作品在高水平的算法表达之外，多了一层内在的语义深度。
